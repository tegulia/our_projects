## Excel版本历史参考

本指南仅作为Excel版本的历史参考，不再推荐使用Excel进行学员管理。如果您仍需要了解Excel版本的使用方法，请参考以下内容。

### Excel版本概述

Excel版本的蔬食料理课程学员管理系统是早期的管理工具，主要通过Excel表格和公式实现学员信息管理、签到记录和课程统计功能。

### 主要功能模块

1. **学员信息表**
   - 学员基本信息记录
   - 报名类型和课时管理
   - 联系方式存储

2. **签到记录表**
   - 每次课程签到情况
   - 课时自动扣减
   - 出勤率统计

3. **课程安排表**
   - 课程日期和主题
   - 授课教师信息
   - 课程进度跟踪

4. **统计分析表**
   - 学员类型分布
   - 课程出勤情况
   - 图表可视化展示

### Excel版本限制

与Web应用相比，Excel版本存在以下限制：

1. **数据安全性**
   - 文件容易丢失或损坏
   - 缺乏自动备份机制
   - 多人协作困难

2. **功能限制**
   - 需要手动更新统计数据
   - 缺乏实时数据同步
   - 操作复杂，易出错

3. **可访问性**
   - 需要安装Excel软件
   - 不支持移动设备访问
   - 无法远程访问数据

4. **用户体验**
   - 界面不够直观
   - 操作步骤繁琐
   - 缺乏交互反馈

### 数据迁移指南

如果您之前使用Excel版本管理学员数据，可以按照以下步骤迁移到Web应用：

1. **导出Excel数据**
   - 打开Excel文件
   - 将各工作表数据导出为CSV格式

2. **准备数据格式**
   - 确保学员ID格式一致
   - 检查日期格式是否标准
   - 验证联系电话格式

3. **手动导入数据**
   - 打开Web应用
   - 通过"添加学员"功能逐个添加学员信息
   - 通过"添加签到记录"功能导入历史签到数据
   - 通过"添加课程"功能导入课程安排

### Excel版本文件结构

Excel版本包含以下工作表：

1. **学员信息表**
   - 列：学员ID、姓名、联系电话、报名类型、剩余课时、备注

2. **签到记录表**
   - 列：签到ID、学员ID、姓名、签到日期、课程序号、扣课类型

3. **课程安排表**
   - 列：课程ID、课程序号、课程日期、课程主题、授课教师

4. **统计分析表**
   - 学员类型统计
   - 课程出勤统计
   - 图表展示区域

### Excel版本公式参考

如果您需要维护Excel版本，以下是一些关键公式：

1. **剩余课时计算**
   ```
   =IF(报名类型="全课",10-SUMIF(签到记录表[学员ID],[@学员ID],签到记录表[扣课时数]),1-SUMIF(签到记录表[学员ID],[@学员ID],签到记录表[扣课时数]))
   ```

2. **出勤率计算**
   ```
   =IF(报名类型="全课",SUMIF(签到记录表[学员ID],[@学员ID],签到记录表[扣课时数])/10,SUMIF(签到记录表[学员ID],[@学员ID],签到记录表[扣课时数]))
   ```

3. **本月签到次数**
   ```
   =COUNTIFS(签到记录表[学员ID],[@学员ID],签到记录表[签到日期],">="&EOMONTH(TODAY(),-1)+1,签到记录表[签到日期],"<="&EOMONTH(TODAY(),0))
   ```
