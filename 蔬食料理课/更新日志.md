# 蔬食料理课程学员管理与统计系统 - 更新日志

## 版本 1.0.0 (2024-01-20)

### 🎉 首次发布

这是蔬食料理课程学员管理与统计系统的第一个Web版本，标志着从Excel版本到Web应用的全面升级。

### ✨ 新增功能

#### 核心功能模块

1. **学员管理**
   - 学员信息添加、编辑、删除功能
   - 支持全课学员和单次学员分类管理
   - 自动课时计算和剩余课时显示
   - 学员信息表格展示和搜索功能

2. **签到记录**
   - 学员签到记录添加、编辑、删除功能
   - 自动课时扣减机制
   - 支持常规签到和补课签到类型
   - 签到历史记录查看和管理

3. **课程安排**
   - 课程信息添加、编辑、删除功能
   - 课程日期、主题和教师信息管理
   - 课程序号自动排序
   - 课程进度跟踪

4. **统计分析**
   - 学员类型分布饼图展示
   - 课程出勤情况柱状图展示
   - 实时统计数据更新
   - 关键指标看板展示

#### 数据管理功能

1. **本地数据存储**
   - 使用浏览器本地存储技术
   - 数据持久化保存
   - 页面刷新后数据不丢失

2. **数据同步**
   - 学员信息与签到记录实时同步
   - 签到后自动更新学员剩余课时
   - 统计数据实时计算更新

#### 用户界面

1. **响应式设计**
   - 适配不同屏幕尺寸
   - 支持桌面和移动设备访问
   - 自适应布局调整

2. **现代化UI**
   - 基于Bootstrap 5.3.0框架
   - 简洁直观的操作界面
   - 友好的用户交互体验

3. **数据可视化**
   - 集成Chart.js图表库
   - 动态图表展示
   - 实时数据更新

### 🛠 技术栈

- **前端框架**：HTML5, CSS3, JavaScript (ES6+)
- **UI框架**：Bootstrap 5.3.0
- **图表库**：Chart.js
- **字体**：Google Fonts - Poppins
- **数据存储**：浏览器本地存储 (LocalStorage)
