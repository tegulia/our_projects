# 蔬食料理课程学员管理与统计系统

## 项目概述

蔬食料理课程学员管理与统计系统是一个专为"一期十次课"教学模式设计的Web应用程序。系统支持全课学员和单次学员的区分管理，自动记录出勤情况并智能计算剩余课时，为蔬食料理课程提供全面的学员管理解决方案。

## 系统功能

### 1. 学员管理
- 添加、编辑、删除学员信息
- 支持全课学员和单次学员分类管理
- 自动跟踪学员剩余课时
- 学员信息包含：ID、姓名、联系电话、报名类型、剩余课时、备注

### 2. 签到记录
- 记录学员每次课程签到情况
- 自动扣减学员课时（常规签到）
- 支持补课记录（不扣减课时）
- 签到信息包含：学员ID、姓名、签到日期、课程序号、扣课类型

### 3. 课程安排
- 管理十次课程的具体安排
- 设置每节课的日期、主题和授课教师
- 支持添加、编辑、删除课程信息

### 4. 统计分析
- 实时统计全课学员和单次学员数量
- 计算本月签到次数和平均出勤率
- 学员类型分布饼图可视化
- 课程出勤情况柱状图分析

## 技术栈

- **HTML5**：语义化页面结构
- **CSS3**：响应式布局与美观样式
- **JavaScript (ES6+)**：交互逻辑与数据处理
- **Bootstrap 5.3.0**：现代化UI框架
- **Chart.js**：数据可视化图表库
- **Font Awesome**：图标库
- **Google Fonts (Poppins)**：字体支持

## 快速开始步骤

1. **下载项目文件**
   - 将整个"蔬食料理课"文件夹下载到本地

2. **打开应用程序**
   - 直接用浏览器打开`index.html`文件即可使用
   - 推荐使用Chrome、Firefox或Edge等现代浏览器

3. **开始使用**
   - 系统已预置示例数据，您可以立即体验所有功能
   - 根据需要添加、编辑学员信息和课程安排
   - 记录学员签到情况，系统将自动更新统计数据

## 系统特点

### 1. 数据持久化
- 使用浏览器本地存储(localStorage)保存数据
- 刷新页面后数据不会丢失
- 关闭浏览器后数据仍然保留

### 2. 响应式设计
- 适配不同屏幕尺寸的设备
- 在手机、平板和电脑上均有良好体验
- 使用Bootstrap框架确保跨设备兼容性

### 3. 用户友好界面
- 直观的标签页导航
- 清晰的表格展示
- 优雅的模态框表单
- 实时数据统计和图表展示

### 4. 智能功能
- 自动计算剩余课时
- 根据报名类型自动设置初始课时
- 签到后自动更新学员剩余课时
- 实时更新统计数据和图表

### 5. 数据可视化
- 学员类型分布饼图
- 课程出勤情况柱状图
- 实时统计卡片展示
